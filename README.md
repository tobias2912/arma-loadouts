# hjemmeside

hosted on Firebase, built with react

live on https://tobiaseilertsen.no/

# how to run

```
npm install -g firebase-tools
```
or
```
curl -sL https://firebase.tools | bash
```

run locally:
```
firebase serve
```
deploy:
```
npm run build
firebase deploy
```
